Correlation analyses between behavioral and biochemical parameters (Pearson’s correlation, n = 10). Data expressed are exploring time of the novel object (s), AChE (nmol/min/mg protein), GPX (U/mg protein), GSH (μg GSH/ μg protein), protein carbonyl (nmol/mg protein), and MDA (nmol/mg protein). ( A) Exploring time of the novel object vs. MDA ( n = 10, r = −0.679, p < 0.001); ( B) AChE vs. MDA ( n = 10, r = 0.676, p < 0.001); ( C) GPX vs. MDA ( n = 10, r = −0.816, p < 0.0001); ( D) GSH vs. MDA ( n = 10, r = −0.658, p < 0.001) and ( E) protein carbonyl vs. MDA ( n = 10, r = 0.871, p < 0.001).
